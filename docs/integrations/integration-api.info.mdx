API to manage integrations with third-party software. Fixed employee sync field definitions are available via the /employee_sync/fields endpoint. In the /integrations (POST) endpoint, users select which fixed fields are active by providing their keys under the sync_data array (which is a sub-property of each SyncType object). Each item in sync_data must be one of the defined selection objects.
